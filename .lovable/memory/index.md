Liquid glass design system for FLock Chat. iOS 26-style frosted glass surfaces over gradient mesh background.

Design: FLock Chat brand. Liquid glass aesthetic. Inter + JetBrains Mono fonts. iOS mobile-first (max-width 430px). Venice-style bottom drawers with glass surfaces. Pro subscription model with gated features.

Glass tokens in index.css: --glass-bg (0.45 white), --glass-bg-strong (0.65), --glass-bg-subtle (0.25). backdrop-filter: blur(40px) saturate(180%). CSS utility classes: .glass, .glass-strong, .glass-subtle, .glass-hover, .glass-overlay.

Background: multi-stop radial gradient mesh (purple, blue, pink) on body with background-attachment: fixed.

Colors: primary blue #3773FF (HSL 220 100% 60%), foreground dark (220 20% 10%). All surfaces are translucent white with backdrop-filter blur.
